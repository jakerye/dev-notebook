## Packages
Example structure
```
mypackage
  __init__.py
  module_a.py
  module_b.py
  mysubpackage
    __init__.py
    module_c.py
    module_d.py
``` 
Example imports
```
import mypackage.module_a
import mypackage.mysubpacakge.module_c
```
