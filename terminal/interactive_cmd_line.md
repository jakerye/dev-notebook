# Fish Plugin

## Usage
```
fish
```

## OSX Install
```
brew install fish
```

## Ubuntu 16.04 Install
```
sudo add-apt-repository ppa:fish-shell/nightly-master
sudo apt-get update
sudo apt-get install fish
chsh -s /usr/bin/fish
```
